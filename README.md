# Profil Kesim Planlayıcı

Metal profil kesimlerini fireyi azaltacak şekilde planlayan basit PWA.

## Kullanım

Uygulama statik dosyalardan oluşur. Yayına alındığında Android Chrome üzerinden açılıp ana ekrana eklenebilir. İlk açılıştan sonra offline çalışır.

## Lokal Çalıştırma

```powershell
python -m http.server 8000
```

Sonra tarayıcıdan açın:

```text
http://localhost:8000
```

## GitHub Pages

Repository ayarlarından:

1. `Settings`
2. `Pages`
3. `Deploy from a branch`
4. Branch: `main`
5. Folder: `/root`
